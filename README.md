1-para baixar o projeto para sua maquina, acesse o terminal do prompt comando e digite e execute o seguinte comando:
git clone https://github.com/igorcesarprojetos/CadastroPessoas.git

2-Usuario padrão para acesso ao front end é:
Usuario: admin
senha: "sem senha"

3-Para rodar o script do banco de dados se em encontra no projeto da  API em:
"seucaminholocal\CadastroPessoas\CadastroPessoas.View\ScriptDataBase\scriptBanco.sql"

4- para rodar o projeto terá que executar a API primeiro no seu visual studio ou VSCode que está em:
"seucaminholocal\CadastroPessoas\CadastroPessoas.sln"

5- Em seguida acessar o front end:
"seucaminholocal\CadastroPessoas\FrontEnd"

6-Caso tenha o angular inslado no seu computador é só rodar:
npm install

6.1-Senão tiver o angular terá que instalar a versão 19
